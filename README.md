# FinanceMaster

## Description
FinanceMaster is a personal finance management application designed to help users manage their finances effectively. With features such as budget tracking, expense categorization, and financial goal setting, FinanceMaster aims to provide a comprehensive solution for financial management.

## Technologies
- **Backend**: Java
- **Database**: MySQL with JDBC
- **Frontend**: HTML5, CSS3, JavaScript
-  I have just learned Spring Boot Initializ, so I have used Spring Boot in this project as well.

## Features
- User-friendly interface for adding and categorizing expenses.
- Budget tracking and visualization of spending habits.
- Goal setting and progress tracking.
- Secure authentication and data encryption for privacy.
- Integration with financial APIs for real-time data updates (optional).

## Installation
1. Clone the repository: `git clone https://github.com/yourusername/FinanceMaster.git`
2. Set up the MySQL database.
3. Configure the JDBC connection.
4. Run the backend server using Java.
5. Open `index.html` in your browser to start using the application.

## Usage
1. Register an account.
2. Log in to access your dashboard.
3. Add and categorize your expenses.
4. Set and track financial goals.
5. View detailed financial reports.
