# FinanceMaster GUI Application

## Introduction

This project aims to provide a simple graphical user interface (GUI) application, FinanceMaster, designed to help users manage their finances effectively. Originally intended as a web application using Java and Spring Boot, the complexity of Spring Boot led me to focus on developing a standalone GUI version first.

FinanceMaster GUI allows users to input their financial details and receive personalized financial advice based on their inputs. It calculates recommended spending percentages for food, emergencies, and leisure activities, considering factors like income, expenses, and optional financial commitments.

## Motivation

I initially planned to develop FinanceMaster as a web-based Java application using web security tools and Spring Boot. However, due to my current learning curve with Spring Boot and web security, I decided to pivot and develop a GUI version first. This approach enables me to provide a functional tool for users while I continue to deepen my understanding of Spring Boot.

## Features

- User-friendly interface for entering financial information.
- Calculation of recommended spending percentages.
- Dynamic handling of additional financial commitments (funeral cover, medical aid, car insurance).
- Error handling for invalid inputs.
- Interactive options on the final advice page ("Exit" or "Play Tic-Tac-Toe").

## Future Plans

As I continue to learn Spring Boot and web security, I plan to integrate FinanceMaster into a web-based application accessible through browsers or mobile devices. This future version will include enhanced features such as user authentication, data security, and a more seamless user experience.

## How to Use

1. Clone the repository to your local machine.
2. Open the project in IntelliJ IDEA or your preferred Java IDE.
3. Run the `FinanceMasterApp.java` file.
4. Follow the on-screen prompts to input your financial details and receive personalized advice.
